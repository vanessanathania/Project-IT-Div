How to find the first non repeated character of a given String?

The answer is d.So the point that strikes me is that it can be done in two ways. For every index i loop i-1 times and check if that character occur ever again. But this is not efficiet: growth of this method is O(N^2) where N is the lenght of the string. Another possible good way could be if i could form a tree or any other ds such that i could order the character based on the weights(the occurence count).This could take me just one loop of lenght N throught the string to form the structure. That is just O(N) + O(time to build tree or any ds)

Here's a very straightforward O(n) solution: def fn(s);order =[] counts ={} for x in s:if x in counts:counts[x]=1 order append(x) for x in order: if counts[x]==1 return x return none. We loop through the string once.When we come across a new character. We store it in counts with a value of 1 and append it to order.When we come across a character we've seen before,we increment its value in counts.Finally,we loop through order until we find a character with a vlue of 1 in counts an return it
